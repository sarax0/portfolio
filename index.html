Enterprise Knowledge Assistant · Local LLM

An enterprise assistant built on a local LLM, serving senior management and cross-domain teams. I designed the system architecture and resource plan, and implemented retrieval across internal documents, databases, and APIs using a Chroma vector store with hybrid retrieval (BM25 + embeddings + reciprocal rank fusion). Network Log & Alarm Intelligence · SysLogs

Multi-component ML project for network fault detection and root-cause analysis, processing millions of log messages ingested from Elasticsearch, with coverage scaled from 5 to 12 network devices. The system surfaces faults earlier and shortens the path from alarm to root cause.

  • Log anomaly detection (LogBERT): adapted the open-source LogBERT transformer to telecom log data, developing the full preprocessing pipeline from scratch (Drain3 template parsing into sequential model inputs); introduced PEFT fine-tuning for incremental learning, cutting retraining time on millions of logs.
  • Novel log detection (first-time-seen): flags log messages outside the model's training distribution to identify novel fault types early.
  • Log-alarm association & co-occurrence: predicts expected logs from alarm occurrences and identifies causal log-alarm relationships within time windows.

Extracted and consolidated configuration templates from production routers into a standard golden-image configuration applied across devices, reducing configuration errors and outages and cutting setup time for new devices.
